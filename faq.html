Frequently Asked Questions

Why is there no light curve data for some of the sources listed in the main page table?
The light curve repository (LCR) provides flux estimates at three different cadences for >1500 sources that are deemed variable in the 10-year Fermi-LAT point source catalog data release 2 (4FGL-DR2), that is, with a variability index above 21.67. While the table in the main page lists all 4FGL-DR2 sources, only the ones with variability index above 21.67 show embedded hyperlinks to the data products of the source.
